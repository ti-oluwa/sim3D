1. First generate a palette for the video
ffmpeg -i videos/simulation.mp4 -vf "fps=15,scale=1200:-1:flags=lanczos,palettegen=max_colors=64" videos/palette.png

2. Generate GIF with palette
ffmpeg -i videos/simulation.mp4 -i videos/palette.png -filter_complex "fps=10,scale=900:-1:flags=lanczos[x];[x][1:v]paletteuse=dither=none" videos/simulation.gif

3. To compress further 
- sudo apt install gifsicle (Linux)
- brew install gifsicle (MacOS)
- choco install gifsicle (Windows)

Then run 

gifsicle -O3 videos/simulation.gif -o videos/simulation_optimized.gif
